# Documentation Map

Use this directory as a map, not a dumping ground.

Documentation ownership:
- Repo-wide product philosophy and representative workflows belong in [PRODUCT_PRINCIPLES.md](/home/auro/code/warcraft_cli/docs/PRODUCT_PRINCIPLES.md).
- Repo-wide analytics and comparison safety rules belong in [SAFE_ANALYTICS_RULES.md](/home/auro/code/warcraft_cli/docs/SAFE_ANALYTICS_RULES.md).

- [ROADMAP.md](/home/auro/code/warcraft_cli/docs/ROADMAP.md) tracks sequencing, current status, and near-term repo priorities.
- `*_PLAN.md` files track provider-specific implementation state, boundaries, and next steps.
- [USAGE.md](/home/auro/code/warcraft_cli/docs/USAGE.md) is the detailed command and behavior reference.
